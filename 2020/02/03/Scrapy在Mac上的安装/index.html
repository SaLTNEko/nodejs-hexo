Scrapy在Mac上的安装

2020-02-03 教程

Contents

  1. #准备工作
  2. #安装Scapy
  3. #测试
  4. #后续

#准备工作

若你的Mac OS系统为10.11或更高版本：
1.重新启动Mac OS
2.重启时按住Command+R，进入Recovery模式
3.选择实用工具里的终端并输入:
csrutil disable;
4.重新启动，回到Mac OS

#安装Scapy

sudo -H pip install Scrapy

#测试

scrapy version

#后续

1.重新启动Mac OS
2.重启时按住Command+R，进入Recovery模式
3.选择实用工具里的终端并输入:
csrutil enable;
4.重新启动，回到Mac OS
